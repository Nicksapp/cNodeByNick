备注：
1.如果要测试文档里面的 Api ，需要安装 Apizza 的 chrome 插件。不然只能浏览文档，（插件主要功能是支持跨域发送请求）

2.只能在 chrome 内核浏览器上面安装插件

3.右上角选择一下 开发环境，选择线上。因为不同的环境调用的不同的 server 地址

4.目前GET请求是正确的，POST请求我估计CNodejs 做了限制，始终返回 token 错误。

5.如果有什么问题，欢迎留言，我在更正^_^
